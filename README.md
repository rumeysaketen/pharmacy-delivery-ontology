# Pharmacy Delivery Ontology

## Description
This ontology models an online pharmacy delivery system.

## Scope
Includes users, medicines, orders, pharmacies, prescriptions, and delivery personnel.

## Concepts
- User
- Medicine
- Order
- Pharmacy
- Prescription
- DeliveryPerson

## Relationships
- placesOrder
- containsMedicine
- deliveredBy
- providedBy

## Example
Ayse places an order (Order001) containing Paracetamol, delivered by Courier1 from PharmacyA.
